# CUBENZ

A 3D roll-a-ball style game built with Unity, featuring 3 levels, a main menu, and a credits screen. Built as part of the Brackeys 'Create with Code' series.

## Gameplay
- Control a cube through each level
- Avoid obstacles and reach the goal
- 3 progressively challenging levels

## Download & Play
Head to [Releases](../../releases/latest) to download the latest version:
- **Android** — Download and install the APK directly on your device
- **Windows** — Download and extract the zip, then run CUBENZ.exe

## Built With
- Unity 3D
- C#

## Controls
| Platform | Action | Input |
|----------|--------|-------|
| PC | Move | WASD / Arrow Keys |
| PC | Quit | Escape |
| Android | Move  |

## Levels
| Level | Description |
|-------|-------------|
| 1 | Introduction to movement and controls |
| 2 | Increased difficulty with more obstacles |
| 3 | Final challenge |
